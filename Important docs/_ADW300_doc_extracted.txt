Need CS
Enable to add a CS checksum byte to the end of the uploaded data packet. 
This option is selected by default for the initial template
Checksum: All the data in the uploaded packet is added to obtain a value, and then the value is divided by 256 to obtain a remainder, which is the CS checksum.
For example, a packet contains this data:
01 00 00 00 01 02 00 00 00 01
If we need CS, then the CS checksum is added to the end:
01 00 00 00 01 02 00 00 00 01 
05
So in the absence of CS, the packet size is a multiple of 5. If we add CS it's a multiple of 5, plus 1 byte.
The function of CS checksum calculation is constructed by C language
:
//pBus[] is the data buffer.
// start is the location of the first data.
// n is the number of bytes computed.
uint8_t CS16(uint8_t* pBus, uint16_t start, uint16_t n) 
{
Uint16_t i;
uint8_t sum = 0;
for (i = start ; i < n+start ; i++)
{
sum = ( sum + pBus[i] ) % 256;
}
return sum;
}
